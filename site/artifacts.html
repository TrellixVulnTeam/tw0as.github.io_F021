ARTIFACTS

Artifacts are items in Genshin Impact that can be equipped on Characters to increase their Stats.
There are 5 types of Artifacts that can be equipped: Flower of Life, Plume of Death, Sands of Eon, Goblet of Eonothem and Circlet of Logos. Only one of each type can be equipped on a character at a time.
All Artifacts have a main affix, commonly known as a main stat, with up to 4 minor affixes, commonly known as sub stats or secondary stats. These affixes can greatly increase the character's strength in combat. Certain artifact types, such as the Flower of Life and Plume of Death, can only come with a specific main stat, while other types can come with one of many main stats.

ADVENTURER

Adventurer is an Artifact Set available at 1-star, 2-star, and 3-star rarities which can be obtained from Midsummer Courtyard, Chests, Investigation, Shiliu (shop), Oceanid, Pyro Regisvine, and Adventure Rank Rewards.

  • 2-Piece Bonus: Max HP increased by 1,000.
  • 4-Piece Bonus: Opening a chest regenerates 30% Max HP over 5s.

LUCKY DOG

Lucky Dog is an Artifact Set available at 1-star, 2-star, and 3-star rarities which can be obtained from Domain of Guyun, Zhang Shun (shop), Chests, Investigation, Anemo Hypostasis, Cryo Regisvine, Cryo Hypostasis, Maguu Kenki, and Perpetual Mechanical Array.

  • 2-Piece Bonus: DEF increased by 100.
  • 4-Piece Bonus: Picking up Mora restores 300 HP.

TRAVELING DOCTOR

Traveling Doctor is an Artifact Set available at 1-star, 2-star, and 3-star rarities which can be obtained from Valley of Remembrance, Chests, Elite Enemies, Investigation, Xigu Antiques (shop), Electro Hypostasis, Geo Hypostasis, Hydro Hypostasis, Thunder Manifestation, and Golden Wolflord.

  • 2-Piece Bonus: Increases incoming healing by 20%.
  • 4-Piece Bonus: Using Elemental Burst restores 20% HP.

BERSERKER

Berserker, also called Blood-Soaked in the Adventurer Handbook, is an Artifact Set available at 3-star, and 4-star rarities which can be obtained from Elite Enemies, Domain Reliquary: Tier II, Domain Reliquary: Tier III, Anemo Hypostasis, Cryo Hypostasis, Cryo Regisvine, Oceanid, Pyro Regisvine, Pyro Hypostasis, Primo Geovishap, Maguu Kenki, Perpetual Mechanical Array, Weekly Bosses, and Chests.

  • 2-Piece Bonus: CRIT Rate +12%
  • 4-Piece Bonus: When HP is below 70%, CRIT Rate increases by an additional 24%.

BRAVE HEART

Brave Heart, also called Heart of Bravery in the Adventurer Handbook, is a 3-star and 4-star Artifact Set obtained from Domain of Guyun, Ridge Watch, and Hidden Palace of Guizang Formula.

  • 2-Piece Bonus: ATK +18%.
  • 4-Piece Bonus: Increases DMG by 30% against opponents with more than 50% HP.

DEFENDER'S WILL

Defender's Will is an Artifact Set available at 3-star and 4-star rarities which can be obtained from Hidden Palace of Zhou Formula, Peak of Vindagnyr, and Slumbering Court.

  • 2-Piece Bonus: DEF +30%
  • 4-Piece Bonus: For each different element present in your own party, the wearer's Elemental RES to that corresponding element is increased by 30%.

GAMBLER

Gambler is an Artifact Set available at 3-star and 4-star rarities which can be obtained from Clear Pool and Mountain Cavern and Peak of Vindagnyr.

  • 2-Piece Bonus: Increases Elemental Skill DMG by 20%.
  • 4-Piece Bonus: Defeating an opponent has 100% chance to remove Elemental Skill CD. Can only occur once every 15s.

INSTRUCTOR

Instructor is an Artifact Set available at 3-star and 4-star rarities which can be obtained from Elite Enemies, Anemo Hypostasis, Electro Hypostasis, Pyro Regisvine, Pyro Hypostasis, Geo Hypostasis, Hydro Hypostasis, Primo Geovishap, Maguu Kenki, Thunder Manifestation, and Golden Wolflord.

  • 2-Piece Bonus: Increases Elemental Mastery by 80.
  • 4-Piece Bonus: Upon triggering an Elemental Reaction, increases all party members' Elemental Mastery by 120 for 8s.

MARTIAL ARTIST

Martial Artist is an Artifact Set available at 3-star and 4-star rarities which can be obtained from Hidden Palace of Zhou Formula and Ridge Watch.

  • 2-Piece Bonus: Increases Normal Attack and Charged Attack DMG by 15%.
  • 4-Piece Bonus: After using Elemental Skill, increases Normal Attack and Charged Attack DMG by 25% for 8s.

PRAYERS FOR DESTINY

PRAYERS FOR ILLUMINATION

Prayers for Illumination is an Artifact Set available at 3-star and 4-star rarities which can be obtained from the Pyro Regisvine, Primo Geovishap, and Pyro Hypostasis.

Resolution of Sojourner is an Artifact Set available at 3-star and 4-star rarities which can be obtained from Midsummer Courtyard, Momiji-Dyed Court, Adventurer Handbook, and Adventure Rank Rewards.

  • 4-Piece Bonus: Incoming elemental DMG increases corresponding Elemental RES by 30% for 10s. Can only occur once every 10s.

ARCHAIC PETRA

Archaic Petra is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Domain of Guyun.

  • 4-Piece Bonus: Upon obtaining an Elemental Shard created through a Crystallize Reaction, all party members gain 35% DMG Bonus for that particular element for 10s. Only one form of Elemental DMG Bonus can be gained in this manner at any one time.

BLIZZARD STRAYER

BLOODSTAINED CHIVALRY

Bloodstained Chivalry is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Clear Pool, Mountain Cavern and and Mystic Offering (crafting bench).

Emblem of Severed Fate is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Momiji-Dyed Court in Inazuma.

  • 4-Piece Bonus: Increases Elemental Burst DMG by 25% of Energy Recharge. A maximum of 75% bonus DMG can be obtained in this way.

GLADIATOR'S FINALE

Gladiator's Finale is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Bosses at World Level 2 and above, Domain Reliquary: Tier I, Domain Reliquary: Tier II, and Mystic Offering (crafting bench).

Husk of Opulent Dreams is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Slumbering Court.

When 6 seconds pass without gaining a Curiosity stack, 1 stack is lost.

OCEAN-HUED CLAM

Ocean-Hued Clam is an Artifact Set available at 4-star and 5-star rarities which can be obtained from Slumbering Court.

  • 2-Piece Bonus: Healing Bonus +15%.
  • 4-Piece Bonus: When the character equipping this artifact set heals a character in the party, a Sea-Dyed Foam will appear for 3 seconds, accumulating the amount of HP recovered from healing (including overflow healing). Only one Sea-Dyed Foam can be produced every 3.5 seconds. Each Sea-Dyed Foam can accumulate up to 30,000 HP (including overflow healing). There can be no more than one Sea-Dyed Foam active at any given time. This effect can still be triggered even when the character who is using this artifact set is not on the field.
